# Discrete Diffusion — MDLM Workshop

A hands-on walkthrough of Masked Diffusion Language Models (MDLM), a generative model for discrete sequences.

## Files

- `mdlm_walkthrough_exercise.ipynb` — notebook with exercises to fill in
- `mdlm_walkthrough_soln.ipynb` — completed solution notebook
- `mdlm_data.py` — data utilities: text8 download, tokenization, and evaluation metrics
- `mdlm_model.py` — DiT-style transformer backbone

## Instructions

Work through `mdlm_walkthrough_exercise.ipynb` and refer to `mdlm_walkthrough_soln.ipynb` as needed.
